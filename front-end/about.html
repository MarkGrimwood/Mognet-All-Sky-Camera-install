Raspberry Pi based all sky camera

Raspberry Pi based all sky camera software by Mark Grimwood (Mognet on Stargazer's Lounge). Based on Pete's all sky camera (discussion thread on SGL)

Images are taken every minute during the daytime and every two minutes at night, and each image is added to a video for that period. The images are date and time stamped, and the stamps also appear in the videos

A history of videos and images is preserved until space on the memory card runs low, after which the oldest period(s) will be deleted to ensure enough space for the next set of captures. Clean up code runs when available space drops below 2Gb at the start of a capture period

The code has been written to be light enough to run on a Pi Zero without stressing it, and the results viewable in a browser on PC, laptop, tablet and mobile phone

Day and night camera modes are automatically catered for and switching occurs at the appropriate times. Image capture and processing is based on a Cron job, and the times are regenerated daily at noon


This project lives on GitHub https://github.com/MarkGrimwood/Mognet-All-Sky-Camera-install
